# [[100단 미세 스크롤 제어 기능을 갖춘 고성능 '초고속 - Planner]]

## 📌 Brief Summary
**Target Role:** 너는 10년 차 이커머스 상품 기획자야. 타겟의 페인포인트와 USP를 분석해.
**Task Context:** 100단 미세 스크롤 제어 기능을 갖춘 고성능 '초고속 메탈 제트 터보팬 핸디 선풍기'의 상세페이지 상품 전략을 수립해 줘. 기존의 획일적인 풍량에 한계를 느끼는 하이테크 얼리어답터 타겟 사양이어야 해....

## 📖 Core Content
* 타겟 페인포인트 분석
1. 정해진 3~4단계 단풍 구조로는 너무 강하거나 약해서 실내외 온도 변화에 따른 유연한 풍량 맞춤이 불가능함.
2. 모터 분당 회전수(RPM)가 낮아 정오의 한여름 아스팔트 열기 속에서는 시원한 바람을 멀리 보내지 못함.
3. 디지털 화면 부재로 사용 도중 전원이 언제 꺼질지 모르는 스마트 기기 유저 특유의 조바심 존재.

* 제품 USP 매칭을 통한 기획 구조
1. 100단 스마트 디지털 스크롤 조작: 조그다이얼을 돌리듯 측면 스크롤을 활용하여 1부터 100까지 풍량을 세밀하고 부드럽게 무단계로 제어하는 혁신성 부각.
2. 항공기 터빈 원리 메탈 블레이드: 강력한 고속 터보 모터 구동 모듈과 내부 금속 가공 역학을 결합하여 직진성으로 뻗어 나가는 강력한 시속 수십 km급 '제트 기류' 구현.
3. 고해상도 원형 LED 대형 화면: 전면에 실시간 풍량 레벨과 배터리 잔량 단위를 명확한 스크린으로 배치하여 시각적 직관성과 테크니컬 프리미엄 가치 제공.
4. 미니멀 일체형 알루미늄 메탈릭 룩: 플라스틱 완구 느낌을 완전히 배제한 매끄러운 원통형 메탈릭 바디 바디 라인으로 소장 가치 극대화.
