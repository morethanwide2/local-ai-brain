# [[프리미엄 알루미늄 캐리어 상세페이지의 비주얼 콘셉트와 - Copywriter]]

## 📌 Brief Summary
**Target Role:** 너는 시각화 전문 디자인 디렉터야. 카피를 시각적으로 연출하는 가이드를 만들어.
**Task Context:** 프리미엄 알루미늄 캐리어 상세페이지의 비주얼 콘셉트와 섹션별 레이아웃 연출 가이드를 작성해 줘. 모던하고 고급스러운 광고 무드가 드러나야 해....

## 📖 Core Content
* 전체 톤앤매너 & 디자인 시스템
- 주조색: 딥 블랙(Dark Slate) 및 메탈릭 실버(Metallic Silver) 테크니컬 컬러 레이아웃
- 폰트: 볼드하고 모던한 산세리프 계열을 메인으로 사용, 장식 요소를 배제하고 자간을 좁혀 고급 가전 광고의 신뢰감 형성

* 섹션별 비주얼 레이아웃 가이드
1. 히어로 섹션: 블랙 스튜디오 배경에 강렬한 탑 라이팅(Top lighting)을 떨어뜨려 캐리어 본체의 알루미늄 리브 라인과 메탈릭한 질감을 극대화. 어두운 공간 속에서 제품이 묵직하게 돋보이는 구도 연출.
2. 내구성 테스트 섹션: 차량이 캐리어 위를 직접 밟고 지나가는 리얼한 현장 사진이나 압축 테스트 기계 내부의 정밀한 측정 컷을 배치하여 시각적 데이터 신뢰성 제공.
3. 내부 수납 디테일: 화이트 앤 다크 그레이 톤의 깔끔한 스튜디오 탑뷰(Top-down) 구도로 내부 파티션을 촬영. 의류, 여권, 화장품 등이 정갈하게 분할 수납된 모습을 기하학적 그리드로 배치.
4. 라이프스타일 연출 씬: 이국적인 유럽 도심의 돌바닥 스트리트 혹은 공항 터미널 배경에서 세련된 스타일링의 모델이 캐리어를 부드럽게 끌고 이동하는 파파라치 컷 느낌의 비주얼을 배치해 실제 사용 시의 세련된 무드 전달.
