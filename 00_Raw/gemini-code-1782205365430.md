# [[Semiconductor Cooling Fan E-commerce 상세페이지 프롬프트 패키지]]

## 📌 Brief Summary
본 프롬프트는 휴대용 반도체 냉각 선풍기(Semiconductor Cooling Fan)의 국내 이커머스(쿠팡, 스마트스토어 등) 상세페이지 제작을 위해 기획된 총 12개 섹션의 완벽한 구조화 가이드라인입니다.[cite: 4]

## 📖 Core Content

### 1. 기본 제품 정체성 및 디자인 원칙 (Product Identity)
*   **외형 특징:** 화이트 바디, 원형 팬 헤드 및 전면 그릴, 중앙 반도체 냉각 플레이트, 콤팩트한 핸들, 접이식 거치 구조, 디지털 디스플레이.[cite: 4]
*   **디자인 무드:** 프리미엄 가전 광고 스타일, 화이트와 아이스 블루 중심의 쿨링 색상 패밀리, 높은 모바일 가독성을 위한 프리텐다드(Pretendard) 타이포그래피.[cite: 4]
*   **배제 요인 (Avoid):** 중국식의 난잡한 레이아웃, 조잡한 가짜 인증 마크 및 할인 정보, 근거 없는 수치(배터리 용량 등) 및 의학적 효과 표기, 만화 같은 질감, 제품 형상을 가리는 과도한 연출.[cite: 4]

### 2. 상세페이지 섹션별 기획 (Section 01 ~ 12)

#### Section 01: 프리미엄 히어로 섹션 (Hero Section)
*   **비주얼:** 극적인 눈 덮인 빙하/빙산 배경. 신선한 눈 속에 부분적으로 묻혀 있거나 서 있는 두 대의 제품, 미세한 성에와 얼음 입자 연출. 콜드 블루-화이트 톤의 차가운 안개 및 얼음 기류 효과 적용.[cite: 4]
*   **카피 구성:**
    *   *상단 카피:* "더위는 바람만으로 부족하다, 냉각이 답이다"[cite: 4]
    *   *메인 헤드라인:* 초강력 반도체 냉각 패드[cite: 4]
    *   *서브 카피:* "버튼 한 번으로 즉시 냉각 모드 ON! / 냉각 플레이트가 피부에 닿는 순간 시원한 냉감을 전해줘 / 뜨거운 여름에도 오아시스 같은 시원함을 느껴보세요."[cite: 4]
*   **하단 영역:** 팬을 볼, 얼굴, 목 주변에 대고 사용하는 한국인 여성 모델의 라이프스타일 포토 카드 3장 배치.[cite: 4]

#### Section 02: 문제 제기 및 니즈 환기 섹션 (Problem / Need Section)
*   **비주얼:** 일반 바람으로는 부족한 뜨거운 여름 상황 연출 (온·냉 대비 레이아웃).[cite: 4]
    *   *좌측:* 강한 햇빛 아래서 더위를 느끼며 이마나 목을 짚고 있는 한국인 여성.[cite: 4]
    *   *우측:* 냉각 패드를 볼이나 목에 대고 안도감을 느끼는 여성 (냉각 플레이트에서 은은한 블루 냉기 아우라 발산).[cite: 4]
*   **카피 구성:**
    *   *상단 라벨:* Problem[cite: 4]
    *   *헤드라인:* 바람만으로는 부족한 여름, / 피부에 닿는 냉각이 필요합니다[cite: 4]
    *   *서브 카피:* "일반 바람은 순간만 시원하고, / 열 오른 볼과 목은 금방 다시 답답해지기 쉽습니다. / 그래서 직접 닿는 냉각 패드가 더 시원하게 느껴집니다."[cite: 4]
*   **하단 카드:** [얼굴 열감이 오래갈 때] [목덜미까지 답답할 때] [바람만으로 아쉬울 때][cite: 4]

#### Section 03: 핵심 기능 요약 섹션 (Feature Summary Section)
*   **비주얼:** 깔끔한 화이트 및 스카이 블루 톤의 기능 오버뷰 페이지. 제품을 중앙 또는 상단에 크게 배치하고, 주변에 깔끔한 라인 스타일의 기능 아이콘들을 조화롭게 정렬 (과도한 장식 배지 지양).[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 한 손에 담긴 여름 냉각 솔루션[cite: 4]
    *   *서브 카피:* "바람, 냉각, 휴대성까지 / 더운 날 필요한 기능을 한 번에 담았습니다."[cite: 4]
*   **기능 포인트 (6가지):** 강력한 바람 / 반도체 냉각 패드 / 디지털 표시창 / 접이식 거치 / 휴대하기 좋은 크기 / 간편한 버튼 조작[cite: 4]

#### Section 04: 냉각 패드 디테일 섹션 (Cooling Plate Detail Section)
*   **비주얼:** 팬 헤드와 중앙 냉각 플레이트의 초대형 클로즈업 샷. 중앙의 메탈 냉각 플레이트에는 응결된 이슬, 성에, 미세한 물방울และ 블루 아이시 하이라이트 효과를 주어 극도의 차가움을 시각화. 플레이트를 가리키는 정교한 기술 콜아웃 라인(지시선) 추가.[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 피부에 닿는 순간, / 차가운 냉감이 전해집니다[cite: 4]
    *   *서브 카피:* "중앙 냉각 패드가 열 오른 피부에 직접 닿아 / 바람만으로 부족했던 시원함을 더해줍니다."[cite: 4]

#### Section 05: 테크니컬 성능 섹션 (Dark Tech Section)
*   **비주얼:** 다크 네이비 / 블랙 배경에 블루 아우라의 공기 흐름 및 냉각 에너지 그래픽 적용. 제품에 드라마틱한 조명을 비추어 팬 헤드, 냉각 플레이트, 디지털 디스플레이를 고급스럽게 강조.[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 작은 바디 안에 담은 / 강력한 냉각 퍼포먼스[cite: 4]
    *   *서브 카피:* "바람과 냉각을 함께 느끼도록 설계된 / 휴대용 쿨링 팬입니다."[cite: 4]
*   **비주얼 요소:** 블루 기류 라인, 모터에서 영감을 받은 미세한 원형 그래픽, 차가운 안개, 프리미엄 가전 라이팅.[cite: 4]

#### Section 06: 송풍 성능 섹션 (Wind Performance Section)
*   **비주얼:** 제품이 작동하며 깨끗하고 선명한 블루 기류를 뿜어내는 모습 시각화. 손에 들고 있거나 데스크 위에 올려진 연출.[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 작지만 강하게, / 더운 공기를 빠르게 날려줍니다[cite: 4]
    *   *서브 카피:* "외출 중에도, 책상 위에서도 / 필요한 순간 바로 시원하게 사용할 수 있습니다."[cite: 4]
*   **포인트 카드:** 강한 송풍 / 간편 조작 / 선명한 표시창 / 실내외 활용[cite: 4]

#### Section 07: 접이식 구조 섹션 (Foldable Stand Section)
*   **비주얼:** 깔끔한 데스크 혹은 미니멀한 블루-화이트 스튜디오 배경. 두 가지 모드의 제품 컷을 동시에 제시.[cite: 4]
    1. 손에 들고 사용하는 세로형 모드 (Handheld upright mode)[cite: 4]
    2. 힌지를 접어 책상 위에 안정적으로 세워둔 모드 (Folded tabletop standing mode)[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 손에 들고, 책상 위에 두고 / 상황에 맞게 사용하세요[cite: 4]
    *   *서브 카피:* "접이식 구조로 외출용 핸디팬은 물론 / 책상 위 쿨링팬처럼 활용할 수 있습니다."[cite: 4]
*   **콜아웃 레이블:** 핸디 사용 / 탁상 거치 / 각도 조절 / 컴팩트 보관[cite: 4]

#### Section 08: 휴대성 강조 섹션 (Portability Section)
*   **비주얼:** 밝고 청량한 여름 라이프스타일 톤앤매너. 깔끔한 의상을 입은 한국인 여성 모델이 토트백에 선풍기를 쏙 넣거나, 야외 활동 중 가볍게 쥐고 있는 내추럴한 모습 연출. 제품이 비현실적으로 커 보이지 않도록 실제 비율 유지.[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 가방에 쏙, / 더운 순간 바로 꺼내세요[cite: 4]
    *   *서브 카피:* "출근길, 여행, 산책, 야외활동까지 / 여름 외출에 부담 없이 챙기기 좋은 사이즈입니다."[cite: 4]

#### Section 09: 다채로운 사용 씬 섹션 (Usage Scene Section)
*   **비주얼:** 정돈된 카드 그리드(Card-grid) 레이아웃을 통해 일상 속 다양한 순간들을 직관적으로 시각화.[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 더운 순간마다 / 시원함을 가까이에[cite: 4]
    *   *서브 카피:* "실내에서도, 외출 중에도 / 필요한 곳에서 바로 사용할 수 있습니다."[cite: 4]
*   **사용처 카드 리스트:** 출근길 / 사무실 책상 / 화장 전후 / 야외 산책 / 캠핑 및 피크닉 / 여행지[cite: 4]

#### Section 10: 상세 디테일 뷰 섹션 (Product Detail Section)
*   **비주얼:** 깨끗한 화이트 배경 위에 제품의 디테일 영역 요소들을 클로즈업하여 배치하고, 얇은 블루 콜아웃 라인(지시선)으로 마킹 (가짜 방수 기능 표기 금지).[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 디테일까지 깔끔한 / 프리미엄 쿨링 팬[cite: 4]
    *   *서브 카피:* "자주 손이 가는 제품일수록 / 형태와 사용 편의성이 중요합니다."[cite: 4]
*   **지시선 레이블:** 냉각 패드 / 전면 그릴 / 디지털 표시창 / 접이식 힌지 / 충전 포트 / 사이드 버튼 / 핸들 바디[cite: 4]

#### Section 11: 제품 구성 및 스펙 섹션 (Package and Specification Section)
*   **비주얼:** 화이트 배경에 연한 블루 구분선을 사용한 모던한 인포메이션 테이블 레이아웃. 측면에 실제 제품 이미지를 단정하게 배치 (상상 속 허구의 액세서리 포함 금지).[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 구매 전 확인하는 / 제품 구성과 기본 정보[cite: 4]
*   **테이블 항목:** 제품명 / 색상 / 충전 방식 / 제품 형태 / 사용 방식 / 구성품[cite: 4]

#### Section 12: 안내 사항 및 최종 구매 유도 섹션 (Notice and Final CTA Section)
*   **비주얼:** 깔끔한 아이스 블루 배경을 바탕으로, 상단의 히어로 제품 비주얼을 다시 한번 배치하여 구매 욕구를 자극하는 마무리 구성.[cite: 4]
*   **카피 구성:**
    *   *헤드라인:* 올여름, 바람보다 더 시원한 / 냉각을 경험해보세요[cite: 4]
    *   *서브 카피:* "얼굴, 목덜미, 손목까지 / 뜨거운 순간 바로 닿는 냉각감."[cite: 4]
