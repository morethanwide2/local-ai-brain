# [[데스크 환경을 깔끔하게 정리하고 거북목을 방지하는 '인 - Planner]]

## 📌 Brief Summary
**Target Role:** 너는 10년 차 이커머스 상품 기획자야. 타겟의 페인포인트와 USP를 분석해.
**Task Context:** 데스크 환경을 깔끔하게 정리하고 거북목을 방지하는 '인체공학적 알루미늄 모니터 받침대(스탠드)'의 상세페이지 상품 기획을 도출해 줘. 공간 활용도에 초점을 맞춰야 해....

## 📖 Core Content
* 타겟 페인포인트 분석
1. 모니터 높이가 낮아 장시간 작업 시 목과 어깨에 통증(거북목 증후군)을 느끼는 직장인 속출.
2. 키보드, 마우스, 필기구 등이 책상 위를 차지하여 유효 작업 공간이 좁아지고 집중력이 떨어짐.
3. 시중의 플라스틱이나 나무 받침대는 모니터 무게를 이기지 못하고 중앙이 휘어지거나 디자인이 투박해 인테리어를 해침.

* 제품 USP 매칭을 통한 기획 구조
1. 고하중 통 알루미늄 바디 내구성: 수십 kg의 대형 모니터나 아이맥을 올려두어도 전혀 휘어짐이 없는 최고급 아노다이징 알루미늄 소재 공정으로 하이엔드 데스크테리어 가치 제공.
2. 하단 키보드 수납 공간 레이아웃: 받침대 하부에 넓은 오픈 유효 공간을 설계하여 작업 후 키보드와 마우스를 쏙 밀어 넣을 수 있는 '데스크 0원 공간 활용성' 입증.
3. 사용자 맞춤형 인체공학 높이 설계: 시선 각도를 정면으로 올려주어 허리와 목을 곧게 펴주는 최적의 인체공학적 서포트 라인을 시각적 수치로 증명.
